🌐 Сервис по построению ортодромии

Интерактивный веб-сервис для точного расчета и визуализации ортодромии (кратчайшего расстояния между двумя точками на поверхности земного эллипсоида). Серверная часть производит геодезические вычисления, а фронтенд мгновенно отображает дугу на динамической карте.

🔥 Ключевые возможности
* Высокоточные вычисления: Расчет промежуточных точек ортодромии на эллипсоиде с помощью библиотеки pyproj.

* Гибкость систем координат: Поддержка различных СК на входе с автоматическим пересчетом в WGS84 для отрисовки:

    + WGS84 (Географическая, EPSG:4326)

    + СК-42 (Пулково 1942, EPSG:4284)

    + Меркатор (Сферический Меркатор, EPSG:3857)

* Интерактивный интерфейс: Удобная боковая панель для ввода координат и настройки количества точек (детализации линии).

* Умная карта: Динамический зум и автоматическое центрирование карты Leaflet под размеры построенного маршрута.

🛠 Технологический стек
* Бэкенд: Python 3, Flask, pyproj

* Фронтенд: JavaScript, Leaflet, HTML5 / CSS3

📂 Структура каталогов

Для корректной работы относительных путей Flask (..frontend/index.html) проект должен быть организован следующим образом:

```
constructions_orthodromy/
├── backend/
│   ├── main.py          # Запуск Flask-сервера и API-роуты
│   └── calculation.py   # Функции расчета ортодромии
└── frontend/
    └── index.html       # Веб-интерфейс и логика карты
```

🚀 Быстрый старт
1. Подготовка окружения и установка

Рекомендуется использовать виртуальное окружение venv:

```Bash
# Создание и активация окружения
python3 -m venv venv
source venv/bin/activate  # Для Linux

# Установка необходимых зависимостей
pip install Flask pyproj
```

2. Запуск приложения

Перейдите в папку с бэкендом и запустите скрипт:

```Bash
cd backend
python3 main.py
```

3. Использование

Откройте браузер и перейдите по адресу:

👉 http://127.0.0.1:5000
