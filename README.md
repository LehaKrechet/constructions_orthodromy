# 🌐 Сервис по построению ортодромии и учету зон запрета

Интерактивный веб-сервис для точного расчета и визуализации ортодромии (кратчайшего расстояния между двумя точками на поверхности земного эллипсоида). Серверная часть производит геодезические вычисления, а фронтенд отображает дугу на динамической карте.

Также присутствует возможность ставить, удалять и редактировать полигоны зон запрета (зон исключения). Часть ортодромии, проходящая через зону запрета, автоматически вычисляется в БД и выделяется на карте красным цветом.

Также есть возможность сохранения ортодромии, и вычесления какиих из сохраненных ортодромий проходят через исследуемую окружность.

## 🔥 Ключевые возможности

* **Высокоточные вычисления:** Расчет промежуточных точек ортодромии на эллипсоиде с помощью библиотеки `pyproj`.
* **Гибкость систем координат:** Поддержка различных СК на входе с автоматическим пересчетом в WGS84 для отрисовки:
    * WGS84 (Географическая, EPSG:4326)
    * СК-42 (Пулково 1942, EPSG:4284)
    * Меркатор (Сферический Меркатор, EPSG:3857)
* **Интерактивный интерфейс:** Удобная боковая панель для ввода координат и настройки количества точек (детализации линии).
* **Умная карта:** Отображение маршрутов и зон с помощью карты Leaflet.
* **Полигоны зон запрета:** Возможность динамически создавать, редактировать и удалять полигоны зон запрета на карте с сохранением геометрии в базе данных.
* **Пространственный анализ (БД):** Определение пересечений линии пути с зонами запрета на уровне СУБД с использованием функций PostGIS (`ST_Intersection`, `ST_Intersects`).

## 🛠 Технологический стек

* **Бэкенд:** Python 3, Flask, pyproj, psycopg2, python-dotenv
* **База данных:** PostgreSQL с расширением PostGIS
* **Фронтенд:** JavaScript, Leaflet, Leaflet.draw, HTML5 / CSS3


🚀 Быстрый старт
1. Подготовка окружения и установка

Рекомендуется использовать виртуальное окружение venv:

```Bash
# Создание и активация окружения
python3 -m venv venv
source venv/bin/activate  # Для Linux

# Установка необходимых зависимостей
pip install -r requirements.txt
```
2. Настройка базы данных

В корне создайте файл .env со следующими переменными для подключения к PostgreSQL/PostGIS:
```Bash
DB_HOST=localhost
DB_NAME=your_db_name
DB_USER=your_db_user
DB_PASS=your_db_password
```
Включение расширения PostGIS
```Bash
CREATE EXTENSION IF NOT EXISTS postgis;
```

Создание таблицы exclusion_zones и orthodromys

```Bash
CREATE TABLE exclusion_zones (
    -- Автоматически инкрементируемый первичный ключ
    id SERIAL PRIMARY KEY,
    
    -- Геометрическое поле для хранения полигонов (двумерные, SRID 4326)
    geom geometry(Polygon, 4326) NOT NULL
);

CREATE TABLE orthodromys (
    id SERIAL PRIMARY KEY,
    orthodromy DOUBLE PRECISION[]
);
-- Создание пространственного индекса GIST для быстрого поиска пересечений
CREATE INDEX exclusion_zones_geom_idx ON exclusion_zones USING gist (geom);


```
3. Запуск приложения

Перейдите в папку с бэкендом и запустите скрипт:

```Bash
cd backend
python3 main.py
```

4. Использование

```Bash
cd frontend/angular_fd
nmp install
npm start
```
